Ruby ❤️ WebAssembly

Execute Ruby on the web.

Artichoke compiles to WebAssembly with Emscripten. We plan to support compiling for wasm32-unknown-unknown targets.

MRI-Compatible

Artichoke targets MRI Ruby 2.6.3.

If Artichoke does not run Ruby source code in the same way that MRI does, it is a bug. MRI compatobility means supporting a C API, if possible.
