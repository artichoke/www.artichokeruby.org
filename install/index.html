Install Artichoke Ruby

Using Cargo

cargo install --git https://github.com/artichoke/artichoke --locked artichoke

Artichoke requires the latest stable Rust, clang, ad bison to build. Build setup instructions can be found on GitHub.

CLI

Build applications with artichoke.

$ artichoke -e "puts RUBY_DESCRIPTION"
artichoke 0.1.0 (2020-04-01 revision 2921) [x86_64-darwin]

REPL

Explore with airb.

$ airb
>>> "Hello world!"
