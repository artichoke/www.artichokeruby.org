安装Artichoke Ruby

Artichoke Ruby可安装在Linux、macOS和Windows上。新的构建将于每夜更新。

尝试以下安装方式:

Nightly构建版本 Docker Cargo

预编译的nightly构建内容

在artichoke/nightly下载预编译的二进制文件。文件支持Linux、Linux/musl、macOS和Windows。

每日的二进制文件包含了Artichoke的最新trunk commit。这些二进制文件也可于ruby-build获取。

以往版本 
$ rbenv install artichoke-dev

$ ruby-build artichoke-dev ~/.rubies/artichoke

Docker镜像

Artichoke的nightly构建会被推送至Docker Hub。镜像的构建基于Ubuntu 18.04、Debian Buster Slim和Alpine 3。

这些每日镜像包含了Artichoke的最新trunk commit。

Docker Hub 
$ docker run -it docker.io/artichokeruby/artichoke airb

使用Cargo

Artichoke使用Rust的软件包管理器Cargo进行构建和安装操作。

使用Cargo编译Artichoke的最新trunk commit。

构建指南 
$ cargo install --git https://github.com/artichoke/artichoke --branch trunk --locked artichoke
