Artichoke：基于wasm技术打造下一代Ruby

Artichoke是用Rust打造的Ruby。它可将Ruby应用打包成单个wasm二进制文件，并在模块化的解释器中运行未经信任的代码。

Artichoke现处于Beta版本。我们欢迎贡献者一起打造基于wasm技术的Ruby。

贡献 安装

当前 v0.1.0-pre.0 · 以往版本

安装

使用rbenv、Docker或Cargo安装Artichoke的nightly构建版本。

Nightly构建版本包含artichoke (ruby) 和 airb (irb) 的二进制文件，Artichoke Ruby的核心和标准库运行时也包含于其中。

阅读安装说明 
$ rbenv install artichoke-dev

$ docker run -it docker.io/artichokeruby/artichoke airb

$ cargo install --git https://github.com/artichoke/artichoke --branch trunk --locked artichoke
